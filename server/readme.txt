1.本项目为一个运动休闲名牌购物网站,一共包含7个网页

***项目开发时在gulp环境中开发,浏览项目请在server文件夹中运行文件,local文件为开发时文件***

index(首页)
login(登录页面)
register(注册页面)
list(商品列表页面)
goods(商品详情页)
car(购物车页面)
flashShop(活动页面)
2.登录注册文件放在mylogin目录中,使用在线接口完成登录和注册功能
3.在登录成功后,index(首页)的头部会记录用户名,在顶部显示
在首页的顶部点击[免费注册]会跳转注册页面,点击购物车会跳转到购物车页面.
首页包括banner的无缝轮播,楼层效果,二级导航,懒加载,加载上部分js文件使用了require模块化
在第一层的耐克专区,点击查看更多可以跳转至商品列表页面
其他页面的top,header,footer,都是由jQuery的load()方法加载首页的相关内容渲染出来
4.list(商品列表页面)
商品列表的商品数据都是由json文件goods.json渲染出来的,goods.json在data文件目录下
使用懒加载渲染,
点击去结算跳转至购物车页面,
点击特惠活动跳转至活动页面
点击加入购物车会把数据传入购物车,点击多次数量叠加
点击图片或查看商品详情,可以跳转至商品详情页
5.goods(商品详情页)
相关商品由接收的cookie数据渲染,有放大镜效果,但是不能在此页直接加入购物车
6.flashShop(活动页)
一个静态页面,有放大镜效果和倒计时效果
startTimeStr:'2019/4/11 00:00:00',//开始时间
endTimeStr:'2019/4/17 23:59:59',//结束时间
7.car(购物车页面)
初始时全部选中所有商品
根据传过来的商品生成总数量和总价,未选中的商品不计算价格
每件商品数量至少为1,点击删除可以删除该商品
